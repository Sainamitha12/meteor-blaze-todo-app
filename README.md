# Meteor Blaze Todo App

A Todo application built using Meteor.js, Blaze, and MongoDB.

## Features

- Add tasks
- Categorize tasks (Work, Personal, Urgent)
- Mark tasks as completed
- Delete tasks
- Drag-and-drop task reordering

## Technologies Used

- Meteor.js
- Blaze Templates
- MongoDB
- JavaScript
- HTML
- CSS
- SortableJS

## Running the Application

### Prerequisites

- Node.js
- Meteor.js

### Steps

1. Clone the repository

```bash
git clone https://github.com/Sainamitha12/meteor-blaze-todo-app.git
```

2. Navigate to the project folder

```bash
cd meteor-blaze-todo-app
```

3. Install dependencies

```bash
meteor npm install
```

4. Start the application

```bash
meteor
```

5. Open the application in your browser

```text
http://localhost:3000
```
